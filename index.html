¿Cómo usar GuitarMain?

📥 Descarga

Lo primero de todo es descargar la aplicación, lo cual puedes hacer aquí: Descargar GuitarMain APK

🎸 Setup de la guitarra

Para utilizar la app necesitas colocar un marcador en tu guitarra física. Sigue los siguientes pasos:

  • 1. Descarga el marcador aquí: Descargar Marcador
  • 2. Imprime el PDF en una hoja DIN A4 (ya tiene el tamaño correcto).
  • 3. Recórtalo. Puedes pegarlo a un trozo de cartón o algo resistente para que no se doble.
  • 4. Pégalo en tu guitarra como se muestra en la imagen, usando el principio del mástil y evitando los trastes. Asegúrate de que la parte más larga mire hacia la cuerda más aguda.

📺 Setup de la aplicación para compartir pantalla

La aplicación utiliza la cámara trasera del móvil con realidad aumentada. Para ver los acordes mientras tocas, puedes usar un espejo o un programa para compartir la pantalla.

Si ya usas uno, no necesitas nada más. Sin embargo, para facilitar el proceso, se ha creado un ejecutable. Sigue estos pasos:

  • 1. Descarga el ZIP aquí: Descargar Aqui
  • 2. Descomprímelo en cualquier carpeta de tu PC.
  • 3. Activa el modo depuración USB en tu móvil (guía aquí).
  • 4. Conecta el móvil por USB al PC y ejecuta GuitarMain.exe.
  • 5. ¡Listo! Deberías ver la pantalla del móvil reflejada en tu PC.
